# 스티커 이미지 어플

## 설치 & 실행

```bash
git clone https://github.com/bini59/sticker.git
cd sticker
npm install
```
위 과정으로 필요 모듈까지 설치를 진행한다.

```bash
npm start
```
이후 http://localhost:3000 에서 확인할 수 있다.
